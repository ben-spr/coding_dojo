Task 1: Hannover License Plates
Description:
The goal of this task is to create a regular expression that captures the pattern of license plates for cars from Hannover, Germany. A Hannover license plate consists of three groups:

'H'
Two random uppercase letters
3 to 4 numbers.
Example of valid license plate: 'HAB1234'

Your task is to create a regular expression that matches Hannover license plates and then separate them into the city (3 numbers) and region (4 numbers).


Task 2: Valid Phone Numbers
Description:
The goal of this task is to create a regular expression that captures the pattern of valid phone numbers. Phone numbers may vary in format, but they generally include the following components:

An optional country code (e.g., +1, 001, 1)
An optional area code within parentheses (e.g., (123), (1234))
The main phone number, which may contain digits, hyphens, and spaces.
Example of valid phone numbers: '+1 (123) 456-7890', '001(456) 789-0123', '555-1234', '123 456 789'

Your task is to create a regular expression that matches valid phone numbers.
